프로젝트시 사용해본 대표기술 Servlet, Ajax, JDBC
담당 역할 병원검색, db설계
구현기능 설명 저는 이번프로젝트에서 병원검색 기능을 맡았습니다.

- 검색어와 진료과목 체크박스로 검색 가능합니다.
- 기관명을 누르면 마커를 찍은 후 기관을 중심으로 맵 이동합니다.

예외처리
- 검색어를 입력 안 할 시 검색어를 입력하라는 문구가 뜨고 검색이 안됩니다.
- DB에 해당기관의 좌표가 없으면 좌표가 없다는 알림창이 뜹니다.

구현과정
건강보험심사평가원_병원정보서비스API를 이용해 병원정보 데이터를 xml파일로 가져온 후
JSONObject클래스를 이용해 Json객체로 변환후 데이터베이스에 저장해 검색기능을 구현하였습니다.
프로젝트 참여 소감 및
문제해결

현재  사회적거리두기  때문에  어쩔  수  없이  비대면으로  수업을  들어야하는  상황에서  프로젝트를  하니  어려운  점도  있지만  지금까지  수업들은  내용을  활용하고  무언가를  만들어냈다는  점이  뿌듯하고  좋은  팀원  분들을  만나서  수월하게  인생  첫  프로젝트를  완성했던  것  같습니다.

프로젝트시 어려웠던점
데이터데이스에 api에서 호출한 데이터를 담는데 길이나 형식이 달라 처음에는 많이 해맸습니다.
하지만 문서를 찾아보고 나와있는 표준 형식에 맞춰서 형식을 바꿔 데이터를 성공적으로 담아냈습니다.

프로젝트시 사용해본 대표기술 Spring, Mybatis, WebSocket, FireBase, jQuery

구현기능 설명 - 채팅기능
WebSocket + sockJs + STOMP 를 이용해 실시간 채팅을 구현하였습니다.
데이터베이스와 빠른 통신을위해 NoSQL인 FireBase를 이용해 채팅데이터를 저장했습니다.

- 메모장기능

-프로젝트 설정
설정을 비동기로 on/off 할 수 있도록 구현했습니다.
프로젝트 대표 이미지를 지정할 수 있습니다.

프로젝트 참여 소감 및
문제해결		 세미프로젝트  때  보다  설계규모도  커지고  사용기술도  많아져서  할  수  있는게  많아져서  더욱  유익한  시간이였습니다.  세미프로젝트때는  자바스크립트를  많이사용하지  못했습니다.  하지만  이번  파이널프로잭트를하며  jQuery를  사용해  자바스크립트를  많이 사용해  자바스크립트  숙련도가  많이  늘었습니다. 좋은  팀원분들  만나서 정말 좋았습니다.  서로  정말  큰  힘이  되었던  거  같습니다.
프로젝트시 어려웠던점
채팅에서  이미지파일  업로드/다운로드  구현중  CORS에러가  발생해  Heroku에서  프록시서버를  배포후
업로드/다운로드 요청을  프록시  서버를  경유시켜  해더에
Access-Control-Allow-Origin : *  을  설정해서  문제를  해결하였습니다.
