프로젝트 상세설명
주요 기능 로그인, 병원검색, 약국검색, 의약품 검색, 스케줄러, 게시판
사용툴 Github - https://github.com/MinHyup1/semiProject
Trello - https://github.com/MinHyup1/semiProject

개발환경 
운영체제 : Windows Os
개발도구 : Eclipse
DBMS   : Oracle DB - SQL Developer
Language : JAVA / SQL / HTML5 / CSS / JAVASCRIPT / JSP
프로젝트시 사용해본 대표기술 Servlet, Ajax
담당 역할/ 기여도 담당역할 : 병원검색
기여도 : 30%
구현기능 설명 저는 이번프로젝트에서 병원검색 기능을 맡았습니다.

- 검색어와 진료과목 체크박스로 검색 가능
- 기관명을 누르면 마커를 찍은 후 기관을 중심으로 맵 이동.

예외처리
- 검색어를 입력 안 할 시 검색어를 입력하라는 문구가 뜨고 검색이 안됨.
- DB에 해당기관의 좌표가 없으면 좌표가 없다는 알림창이 뜬다.

구현과정
건강보험심사평가원_병원정보서비스API를 이용해 병원정보 데이터를 xml파일로 가져온 후
JSONObject클래스를 이용해 Json객체로 변환후 데이터베이스에 저장해 검색기능을 구현하였습니다.
